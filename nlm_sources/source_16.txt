# Code Typing Practice - コード分析レポート
## 基本情報
- リポジトリ: susumuiwamoto1717-stack/vibe-22-typing-game
- 作者: susumuiwamoto1717-stack
- フレームワーク: なし（Vanilla JS）
- アーキテクチャ: IIFE + グローバルモジュールパターン
- 規模: 中規模（ファイル数: 14）
- 複雑度スコア: 3/5
- 使用言語: JavaScript, CSS, HTML

## アプリ概要
プログラミング記号・HTML/CSS・JavaScript・Ruby等のコードスニペットをタイピング練習できるWebアプリ。仮想キーボード表示・指ガイド・WPM計測・ダッシュボード（カレンダー式進捗管理）・Ruby/Git学習モードを搭載している。

## 設計の鍵（Design Keys）
1. 状態管理パターン（stateオブジェクトに全アプリ状態を集約し画面遷移を制御）
2. イベント駆動アーキテクチャ（keydownイベントによるリアルタイムタイピング判定）
3. データ駆動設計（SNIPPETSオブジェクトでモード別タイピング問題を構造化管理）

## 学び（Learnings）
1. 状態管理パターン（中央集権的stateオブジェクトによる画面遷移制御）
2. イベント駆動プログラミング（keydownハンドラによるリアルタイム入力処理）
3. 関心の分離（データ/ロジック/UI/表示の4層分離）
