# 推しスタ⭐ - コード分析レポート
## 基本情報
- リポジトリ: tsukakoshi777/motivation-streaming-app
- 作者: tsukakoshi777
- フレームワーク: Ruby on Rails 7.0 + Bootstrap
- アーキテクチャ: MVC + Survey/Goal Domain Model + Decorator Pattern
- 規模: 中規模（ファイル数: 156）
- 複雑度スコア: 3/5
- 使用言語: Ruby, HTML, JavaScript, SCSS, Dockerfile, Shell

## アプリ概要
配信者のモチベーション維持を支援するアプリ。アンケート形式で配信活動を診断し、成長目標（Goal）と日々の輝き（Spark）を記録して自己成長を可視化する。

## 設計の鍵（Design Keys）
1. Draperによるデコレーターパターンの活用
2. Survey→SurveyProfile→SurveyResponse→SurveyResult→Goalという多段階ドメインモデル
3. Sorceryによるカスタム認証 + CarrierWave + AWS S3による画像管理

## 学び（Learnings）
1. 複雑なドメインモデル設計（Survey系の多テーブル構成）
2. Draperデコレーターによるビューロジック分離
3. Turbo Streamによるインライン編集（Sparksの作成・編集・削除）

## BUD分析
- ボトルネック数: 2
- 不必要な作業: 1
- 重複する作業: 1

## セキュリティ懸念
1. dotenv-railsがproductionグループにも含まれている（本番で.envファイル依存）
2. CarrierWaveのアップロード設定でファイルサイズ制限が見当たらない
3. SurveyProfileのネストルーティングでの認可チェックが不明
