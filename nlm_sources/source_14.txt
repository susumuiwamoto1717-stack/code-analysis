# サクノミ（NighTrip） - コード分析レポート
## 基本情報
- リポジトリ: common-member/NighTrip
- 作者: common-member
- フレームワーク: Ruby on Rails 7.2 + Tailwind CSS + Ransack + Geocoder
- アーキテクチャ: MVC + Search/Ranking + Diagnosis Feature + Google Maps Integration
- 規模: 中規模（ファイル数: 292）
- 複雑度スコア: 3/5
- 使用言語: HTML, Ruby, JavaScript, Dockerfile, Shell, CSS

## アプリ概要
夜景スポット共有・検索アプリ。ユーザーが夜景スポットを投稿し、地域・雰囲気・タグで検索・診断できる。ブックマークランキングやGoogle Maps連携機能を備える。

## 設計の鍵（Design Keys）
1. Ransackによる多条件検索 + Geocoderによる住所→緯度経度変換
2. 診断機能（Diagnosis）による条件マッチングとレコメンデーション
3. ブックマーク数ベースのランキング機能（スポット・ユーザー両方）

## 学び（Learnings）
1. Ransack gemによる柔軟な検索フォーム実装
2. Geocoder gemによる住所のジオコーディングとGoogle Maps連携
3. Claude Code連携のプロジェクト設定（.claude/ディレクトリにスキル・ルール・エージェント定義）

## セキュリティ懸念
1. autocompleteアクションでSQLインジェクション対策が不十分（ILIKE + 直接パラメータ埋め込み）
2. Spotモデルのransackable_attributesが比較的広く公開されている
3. coverageディレクトリ（テストカバレッジ結果）がリポジトリにコミットされている
